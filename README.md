## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd Authbot
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To use the bot, you need to have a Discord bot token. Set it in your environment variables or a `.env` file.

Run the bot using:
```
npm start
```
